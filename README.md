# AI Agent – LangChain · LangGraph · Gemini AI

This project is my **first fully working AI Agent**, built using **LangChain**, **LangGraph**, and **Gemini AI**. It showcases an end-to-end agentic workflow with reasoning, state management, and tool interaction.

The agent can run **multi-step tasks**, make decisions, and execute actions through a structured graph-driven architecture. This project represents my foundation in building modern, production-style autonomous AI systems.
